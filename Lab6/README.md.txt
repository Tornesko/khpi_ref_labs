# Лабораторна робота № 6: Дублювання коду

## Мета:

Провести рефакторинг коду з проблемою «Дублювання коду» для покращення його читабельності та підтримуваності.

## Завдання 1

### Опис проблеми:
У вихідному коді було два методи для обчислення загальної вартості з податком та без податку, які мали однакову логіку для розрахунку знижки. Це призводило до дублювання коду.

### Рефакторинг:
Спільну логіку було виділено в окрему функцію `calculate_price`, яка тепер використовується в обох методах для обчислення загальної вартості.

## Завдання 2

### Опис проблеми:
У класі DataAnalyzer було дублювання коду для обчислення загальної суми та кількості елементів. Методи, що виконували ці обчислення, містили однакову логіку.

### Рефакторинг:
Спільну логіку для обчислення мінімального та максимального значення було виділено в окрему функцію `calculate_min_max`, яка тепер використовується в обох методах.

## Висновок

Рефакторинг дозволив зменшити дублювання коду, зробивши його більш чистим та зрозумілим. Це покращило підтримуваність коду та зменшило ймовірність помилок.
